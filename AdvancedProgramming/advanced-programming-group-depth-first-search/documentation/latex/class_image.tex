\doxysubsection{Detailed Description}
Represents an image, encapsulating data and operations for image processing. 

The \doxyref{Image}{p.}{class_image} class is designed to manage and manipulate image data. It supports loading and saving images, accessing image properties (such as width, height, and color channels), and performing basic image processing operations like conversion to grayscale.
