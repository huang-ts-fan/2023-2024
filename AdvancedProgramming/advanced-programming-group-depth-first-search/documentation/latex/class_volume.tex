\doxysubsection{Detailed Description}
Manages and processes 3D volumetric data. 

The \doxyref{Volume}{p.}{class_volume} class is designed to handle 3D data, offering functionalities to load volumetric data from a directory containing image slices, convert this data for processing, and manipulate or analyze the volume through various operations.
