Creates a heatmap showing the probability of a given postcode being affected by the impact event.

Parameters:

probabilitypandas.DataFrame

A DataFrame containing postcodes and their associated impact probabilities. It contains the columns ‘postcode’ and ‘probability’.

populationdict

A dictionary containing population statistics.

latlongfilestr, optional

Path to the CSV file containing latitude and longitude data for each postcode. The CSV file should have columns ‘Postcode’, ‘Latitude’, and ‘Longitude’.

Returns:

fmap: folium.Map

A Folium map object with the heatmap overlay.

Example

>>> probability_df = pd.DataFrame({
...     'postcode': ['CV130AA', 'CV130AE'],
...     'probability': [0.3, 0.1]
... })
>>> population_dict = {'mean': 276347.9, 'stdev': 161686.5}
>>> heatmap = plot_heatmap(probability_df, population_dict)
>>> heatmap.save('heatmap.html')

deepimpact.mapping.plot_meteors(tail=True, filename='/Users/aov23/Documents/ACSE/Git/acs-deepimpact-ceres/deepimpact/../resources/impact_parameter_list.csv')[source]

This function reads a CSV file containing meteor impact parameters, calculates the outcome of each impact, and plots the resulting damage zones on a map.

Parameters:

tailbool, optional

Whether or not to plot the tail of the asteroids. Default is True.

filenamestr, optional

Path to the CSV file containing impact parameters. The CSV file should have columns for ‘radius’, ‘angle’, ‘velocity’, ‘strength’, ‘density’, ‘entry latitude’, ‘entry longitude’, and ‘bearing’. Default path is set to ‘resources/impact_parameter_list.csv’.

Returns:

fmap: folium.Map

A Folium map object with plotted meteor impacts.

Example:

>>> impact_map = plot_meteors()
>>> impact_map.save('meteor_impacts.html')
