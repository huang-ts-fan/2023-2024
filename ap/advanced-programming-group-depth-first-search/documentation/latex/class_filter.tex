\doxysubsection{Detailed Description}
A collection of static methods for applying various image processing techniques. 

The \doxyref{Filter}{p.}{class_filter} class offers a suite of static methods for performing operations related to color correction, blurring, and edge detection on both 2D images and 3D volume data. This class acts as a centralized framework to facilitate easy application of complex image processing techniques. It is designed to be modular and easily extensible for integrating additional image processing functions in the future.\doxysubsubsection{Features\+:}\label{class_filter_autotoc_md0}

\begin{DoxyItemize}
\item Colour Correction\+: Includes methods for grayscale conversion, brightness adjustment, histogram equalization in various color spaces, and application of thresholding techniques.
\item 2D Blur Functions\+: Provides box blur, Gaussian blur, and median blur functionalities to apply on 2D images, useful for noise reduction and image smoothing.
\item Edge Detection\+: Implements edge detection algorithms using operator templates such as Sobel, Prewitt, and Roberts Cross, essential for feature detection and image analysis.
\item 3D Blur Functions\+: Extends blurring techniques to 3D volumetric data, accommodating applications in medical imaging and scientific visualization.
\end{DoxyItemize}

Each method is static, allowing direct invocation on the {\ttfamily \doxyref{Filter}{p.}{class_filter}} class without needing an instance, simplifying the application of multiple filters on the same image or volume data.
